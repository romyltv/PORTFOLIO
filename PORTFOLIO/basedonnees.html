Conception et implémentation d'une base de données

2025-2026

Description du projet

Ce projet consistait à concevoir une base de données, la structurer puis l’exploiter à travers des requêtes. Le projet tournait autour de la créatioon d'une base de données pour un club de voile, comprenant des informations sur les membres, les bateaux, les compétitions et les résultats... Nous avons dû transcrire les spécifications données en schéma relationnel puis implémenter les tables nécessaires. Enfin, nous avons du peupler la base avec des données fictives et rédiger des requêtes SQL pour répondre à des demandes spécifiques. Nous avons réparti le travail équitablement.

Objectifs : Concevoir une base de données, structurer les données, réaliser des requêtes

Techniques et savoir-faire acquis : SQL, modélisation de données, requêtes

Contexte

Le projet a été réalisé en sql avec PostgreSQL, en duo à l’IUT2, sur une durée de plusieurs semaines.

Compétences développées

  • Compétence 1 : Gestion des données
    • Concevoir une base relationnelle
    • Interroger une base de données
    • Assurer la cohérence des données
  • Compétence 2 : Optimisation
    • Structurer les données efficacement
