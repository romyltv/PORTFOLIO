🤖 SAE Chatbot - Intelligence Artificielle

SAE 1.05 | 2024-2025

📝 Description

Cette SAE avait pour objectif de concevoir et développer un chatbot intelligent capable de répondre aux questions des utilisateurs de manière pertinente et contextuelle. Le projet a permis d'explorer les concepts fondamentaux de l'intelligence artificielle et du traitement du langage naturel (NLP).

🎯 Objectifs du projet

  • Comprendre les bases de l'IA et du machine learning
  • Implémenter un système de chatbot fonctionnel
  • Traiter le langage naturel pour interpréter les requêtes utilisateur
  • Générer des réponses pertinentes et contextualisées
  • Tester et optimiser la performance du chatbot

🛠️ Technologies utilisées

Python NLP NLTK Machine Learning JSON

💡 Compétences acquises

  • Programmation Python avancée : Utilisation de bibliothèques spécialisées
  • Traitement du langage naturel : Tokenization, stemming, vectorisation
  • Algorithmie : Implémentation d'algorithmes de matching et de scoring
  • Structures de données : Utilisation de dictionnaires et listes pour la gestion des données
  • Debugging et optimisation : Amélioration des performances du chatbot

👥 Travail en groupe

Ce projet a été réalisé en équipe de 3 personnes. La collaboration s'est basée sur une répartition des tâches claire :

  • Personne 1 : Recherche et documentation sur NLP
  • Personne 2 : Implémentation du moteur de réponse
  • Personne 3 (moi) : Interface utilisateur et intégration

La communication au sein du groupe a été essentielle pour garantir une cohésion entre les différents modules développés en parallèle.

🎓 Contribution personnelle

J'ai été responsable du développement de l'interface utilisateur et de l'intégration des différents modules développés par l'équipe.

  • Création d'une interface CLI (Command Line Interface) conviviale
  • Intégration des modules NLP et réponse
  • Gestion des entrées/sorties utilisateur
  • Tests de l'ensemble du système et rapports de bugs
  • Documentation du code et guide utilisateur

⚙️ Comment utiliser le chatbot

Le chatbot peut être lancé via la ligne de commande et fonctionne en mode conversationnel. L'utilisateur entre sa question et le chatbot répond en temps réel.

✅ Résultats obtenus

Le projet a été évalué satisfaisamment. Le chatbot est capable de :

  • Répondre à plus de 50 questions différentes
  • Adapter ses réponses selon le contexte
  • Apprendre de nouvelles réponses (fonctionnalité additionnelle)
  • Gérer les cas d'incompréhension avec des messages appropriés

🔄 Améliorations futures

  • Utiliser des modèles de deep learning plus avancés (BERT, GPT)
  • Intégrer une base de données pour persister les conversations
  • Développer une interface web pour améliorer l'accessibilité
  • Implémenter l'apprentissage continu avec feedback utilisateur

🎉 Conclusion

Ce projet m'a permis de comprendre les bases de l'intelligence artificielle et du traitement du langage naturel. C'est une expérience enrichissante qui m'a renforcé ma passion pour ce domaine et m'a montré les possibilités infinies de l'IA dans le développement logiciel.
