Développement d’un site pour un client

2026

Description du projet

Ce projet consistait à concevoir et développer un site web répondant à une demande client. Il fallait analyser les besoins, proposer une solution adaptée et réaliser une interface fonctionnelle. Nous avons d'abord eu besoin de réaliser une maquette du site, faire valider par le client, puis de développer les différentes pages en respectant les consignes . Nous avons réparti le travail, chacun s'occupant d'une partie du site, puis nous avons fait une mise en commun pour harmoniser le tout.

Objectifs : Comprendre les attentes d’un client, concevoir une application web complète, développer une interface utilisateur fonctionnelle, travailler en équipe sur un projet concret

Techniques et savoir-faire acquis : Développement web (HTML/CSS), analyse des besoins, organisation de projet, tests et validation

Contexte

Le projet a été réalisé en html et css, en trio, à l’IUT2, sur une durée de plusieurs semaines.

Compétences développées

  • Compétence 1 : Développer une application
    • Implémenter une solution conforme aux besoins
    • Développer une interface utilisateur
    • Tester et valider une application
  • Compétence 2 : Conduire un projet
    • Identifier les besoins du client
    • Organiser le travail en équipe
  • Compétence 3 : Travailler en équipe
    • Collaborer efficacement
    • Communiquer avec les membres du groupe
